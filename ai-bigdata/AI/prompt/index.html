## Skills
- Spring Boot 项目重构与模块拆分

- 接口与领域模型重构

- 单元测试与重构安全保障

## Background（可选）
- 当前项目类型：{单体应用 / 微服务}
- Spring Boot 版本：{如 2.x / 3.x}
