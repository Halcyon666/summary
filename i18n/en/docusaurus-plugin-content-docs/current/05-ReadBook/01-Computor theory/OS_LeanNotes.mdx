## OS 主要功能

1.  OS作为用户和计算机硬件系统之间的接口
2.  OS作为计算机系统资源的管理者
3.  OS实现了对计算机资源的抽象

## 操作系统的发展过程

1.  未配置操作系统的计算机
2.  单道批处理系统
3.  多道批处理系统
    -   多道、宏观上并行，微观上串行。
4.  分时系统
    -   多路性、独立性、及时性、交互性
5.  实时系统
    -   工业控制系统
    -   信息查询系统
    -   多媒体系统
    -   嵌入式系统
6.  微机操作系统的发展
    -   单用户单任务
    -   单用户多任务
    -   多用户多任务
