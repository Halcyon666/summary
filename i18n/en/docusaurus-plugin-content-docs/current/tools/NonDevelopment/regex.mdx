# Regex Cheatsheet

## Remove Duplicate Lines

```shell title="Find what"
^(.*)(\r?\n\1)+
```
```shell title="Replace with"
$1
```

## Match Time

```shell title="Find what"
\d{1,2}:\d{2}(?::\d{2})?
```
