## PicGo Image Upload Issue

When using PicGo(app) or PicList, if you encounter error `"message": "getaddrinfo ENOTFOUND smms",` 

![image-20240307234840371](https://s2.loli.net/2024/03/07/wNRUFiyEqLblzJv.png)



Just switch to `PicGo-Core(command line)`, and modify `～/.picgo/config.json` file:

```json
{
  "picBed": {
    "uploader": "smms",
    "current": "smms",
	"smms": {
      "token": "put your token here"
    }
  },
  "picgoPlugins": {}
}
```

## U30Air SMB Sharing Issue

Solution (Enable insecure guest logons), via Local Group Policy (Suitable for Win10 Pro/Enterprise).

1. `Win + R` → Enter `gpedit.msc` → Enter.
2. Path:
    
    ```
    Computer Configuration → Administrative Templates → Network → Lanman Workstation
    
    ```
    
3. Find policy:
    
    **Enable insecure guest logons**
    
4. Double click → Select **Enabled** → OK.
5. Run `gpupdate /force` to refresh policy.


## Win10 File Copy or Delete Issue

Issue: Time remaining: Calculating... <br/>
Solution: Execute in CMD Administrator mode, then restart computer.
```cmd
sfc /scannow
```

Result after execution: Windows Resource Protection found corrupt files and successfully repaired them.<br/>
For online repairs, details are included in the CBS log file located at windir\Logs\CBS\CBS.log.<br/>
For example C:\Windows\Logs\CBS\CBS.log. For offline repairs, details are included in the log file provided by the /OFFLOGFILE flag.

![20250927015301](https://s2.loli.net/2025/09/27/p5vbtsFy1wqANcz.png)

If still ineffective, use WiseForce Deleter to delete the folder.

## Win10 Disable Updates

```cmd
gpedit.msc
```

In the "Local Group Policy Editor" window's left panel, navigate to: "Computer Configuration" > "Administrative Templates" > "Windows Components" > "Windows Update".
Change "Remove access to use all Windows Update features" to Enabled.
Change "Configure Automatic Updates" to Disabled.

:::info
How to Enable Updates <br/>
Change "Remove access to use all Windows Update features" to Disabled.<br/>
Change "Configure Automatic Updates" to Enabled.
:::
