## Common Shell Commands Collection

### Quick File Writing
1. Method One
```shell script
	cat > a.txt << 'EOF'
	afasdf
	EOF
```

2. Method Two

```shell script
tee a.txt <<- 'EOF'
	afsafdsf
	EOF	
```

### Linux System Scheduled Network Time Synchronization

[Reference Article](https://www.cnblogs.com/jichuang/p/13542177.html)

```shell
# Set scheduled task
crontab -e 
*/5 * * * * ntpdate time.nist.gov

# View scheduled task list
crontab -l

# View scheduled task logs
tail -f /var/log/cron

# Several time servers
ntpdate time.nist.gov
ntpdate time.nuri.net
ntpdate 0.asia.pool.ntp.org
ntpdate 1.asia.pool.ntp.org
ntpdate 2.asia.pool.ntp.org
ntpdate 3.asia.pool.ntp.org

```

[Crontab scheduled task parameter meanings](https://tool.lu/crontab)
