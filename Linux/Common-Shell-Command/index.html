linux系统 定时同步网络时间​

参考文章

# 设置定时任务
crontab -e 
*/5 * * * * ntpdate time.nist.gov

# 查看定时任务列表
crontab -l

# 查看定时任务日志
tail -f /var/log/cron

# 几个时间服务器
ntpdate time.nist.gov
ntpdate time.nuri.net
ntpdate 0.asia.pool.ntp.org
ntpdate 1.asia.pool.ntp.org
ntpdate 2.asia.pool.ntp.org
ntpdate 3.asia.pool.ntp.org
