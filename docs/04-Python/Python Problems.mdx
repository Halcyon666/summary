## Python 使用问题

### pip 源设置 

在 `C:\Users\xxx\pip`下创建 `pip.ini`文件，内容如下


```ini
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
extra-index-url =
    https://mirrors.aliyun.com/pypi/simple/
    https://mirror.baidu.com/pypi/simple
```

> [原文地址](https://zhuanlan.zhihu.com/p/623325525?utm_id=0&wd=&eqid=f10d0e060019cf940000000665767cd2)

### 如何使用python虚拟环境

- **使用 venv**

```shell
python3 -m venv myenv
```

- **使用virtualenv**

```shell
pip install virtualenv
virtualenv myenv
```

- **激活虚拟环境**
