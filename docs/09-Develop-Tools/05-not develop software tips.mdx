## PicGo 上传图片问题

当使用PicGo(app) 或者 PicList时，出现如下报错 `"message": "getaddrinfo ENOTFOUND smms",` 

![image-20240307234840371](https://s2.loli.net/2024/03/07/wNRUFiyEqLblzJv.png)



只需切换到`PicGo-Core(command line)`， 请修改 `～/.picgo/config.json`文件，

```json
{
  "picBed": {
    "uploader": "smms",
    "current": "smms",
	"smms": {
      "token": "put your token here"
    }
  },
  "picgoPlugins": {}
}
```

## 在vscode中

### 使用typora中安装的`PicGo-Core`

1. 在vscode中安装PicGo插件
2. 在vscode中设置PicGo的Picgo: Config Path为 `C:\Users\xxx\.picgo\config.json`

![](https://s2.loli.net/2025/09/20/s8rYEGoZlyJe3US.png)

##  U30Air SMB共享问题

解决办法（启用来宾 SMB 登录）,通过本地组策略（适合 Win10 专业版/企业版）

1. `Win + R` → 输入 `gpedit.msc` → 回车。
2. 路径：
    
    ```
    计算机配置 → 管理模板 → 网络 → Lanman 工作站
    
    ```
    
3. 找到策略：
    
    **启用不安全的来宾登录 (Enable insecure guest logons)**
    
4. 双击 → 选择 **已启用** → 确定。
5. 运行 `gpupdate /force` 刷新策略。
