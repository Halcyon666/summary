# 正则大全

## 去除重复行

```shell title="查找目标"
^(.*)(\r?\n\1)+
```
```shell title="替换为"
$1
```

## 匹配时间

```shell title="查找目标"
\d{1,2}:\d{2}(?::\d{2})?
```
