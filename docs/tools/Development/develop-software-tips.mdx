## VS Code中使用`git bash` terminal

在vscode设置界面，搜索`terminal.integrated.profiles.windows`,然后edit in setting.json.

![image-20250921132552404](https://s2.loli.net/2025/09/21/ahpwFIRPrqTKZ9z.png)

在json文件中搜索`terminal.integrated.profiles.windows`,修改`terminal.integrated.profiles.windows`对应的值，并追加一个属性`terminal.integrated.defaultProfile.windows`。

```json
"terminal.integrated.profiles.windows": {
  "PowerShell": {
    "source": "PowerShell",
    "icon": "terminal-powershell"
  },
  "Command Prompt": {
    "path": [
      "${env:windir}\\Sysnative\\cmd.exe",
      "${env:windir}\\System32\\cmd.exe"
    ],
    "args": [],
    "icon": "terminal-cmd"
  },
  "Git Bash": {
    "path": "D:\\software\\Git\\bin\\bash.exe",
    "icon": "terminal-bash"
  }
},
"terminal.integrated.defaultProfile.windows": "Git Bash"

```

## vscode 取消AI助手提示
- 对所有工程有效：
在设置中搜索`inlineSuggest.enabled`,然后取消勾选即可。

- 如何设置对单个工程有效？在哪创建文件？
在项目根目录创建 `.vscode/settings.json` 文件，添加如下内容：

```json
{
    "editor.inlineSuggest.enabled": false
}
```

## 在vscode中配置java环境

这是最推荐的方式，因为它能同时搞定“执行程序”和“配置文件”。

1. 按 `Ctrl + Shift + P`。
2. 输入 **"Open User Settings (JSON)"** 并回车。
3. 在配置文件中加入以下内容（注意替换你的实际路径）：

```json
{
    "git.autofetch": true,
    "diffEditor.ignoreTrimWhitespace": false,
    "diffEditor.hideUnchangedRegions.enabled": true,
    "git.confirmSync": false,
    "files.autoSave": "onFocusChange",
    "workbench.colorTheme": "Solarized Light",
    "editor.minimap.sectionHeaderFontSize": 10,
    "security.workspace.trust.untrustedFiles": "open",
    "redhat.telemetry.enabled": true,
    // 配置 Java Home (指向你的 JDK 根目录)
    // 注意：Windows 路径里的斜杠要用双斜杠 \\ 或反斜杠 /
    "java.jdt.ls.java.home": "D:\\software\\Java\\jdk-17.0.7",
    // 配置 Gradle Home (指向你的 Gradle 安装目录)
    // 如果你用 wrapper (gradlew)，这一行其实可以不配，但配了更稳
    "java.import.gradle.home": "D:\\software\\gradle-8.7",
    // 可选：指定 Gradle 运行时用的 Java 版本（通常和项目一致）
    "java.import.gradle.java.home": "D:\\software\\Java\\jdk-17.0.7",
    // 1. 指定 Maven 的可执行文件路径 (注意指向 bin 目录下的 mvn 或 mvn.cmd)
    "maven.executable.path": "D:\\software\\apache-maven-3.8.7\\bin\\mvn.cmd",
    // 2. 关键！指定你的 settings.xml 路径 (为了读取阿里云镜像配置)
    "java.configuration.maven.userSettings": "D:\\software\\apache-maven-3.8.7\\conf\\settings.xml",
    // 3. (可选) 如果你的 Maven 需要特定的 JDK 运行
    "maven.terminal.useJavaHome": true
}
```
