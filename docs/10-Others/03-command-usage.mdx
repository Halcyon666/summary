# Bash or PS Command Usage

## 查看目录结构

```powershell
tree /f
```

## 查看端口占用

```powershell
netstat -ano | findstr :8080
```

## 杀掉占用端口的进程

```powershell
taskkill /PID 1234 /F
```

## 临时设置Java Home为另一个路径
```powershell
$env:JAVA_HOME="D:\software\Java\jdk-17.0.7"
$env:PATH="$env:JAVA_HOME\bin;$env:PATH"

./gradlew test --daemon
```
